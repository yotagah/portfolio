Neeblox

Neeblox is a puzzle game I developed solo. Players place Tetris-like pieces on the board and match blocks of the same color to complete objectives and clear the grid. The game features progressively challenging puzzles and mechanics that reward strategic placement and color combinations.

Tic Tac Toe 3D

Tic Tac Toe 3D is a small game I developed over a weekend using HTML5. It reimagines the classic Tic Tac Toe game in a 3D environment and supports up to six players, adding an extra strategic dimension to the traditional gameplay.
